# Fast-Food-Fast
Project Overview Fast-Food-Fast is a food delivery service app for a restaurant.  Required Features Users can create an account and log in. The admin is be able to view all orders or a specific order and add,edit or delete the fast-food items i.e : See a list of orders Accept and decline orders Mark orders as completed A user should be able to see a history of ordered food
